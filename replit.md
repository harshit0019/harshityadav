# Adeola Adeoti Portfolio

## Overview
This is a Next.js portfolio website featuring modern animations with Framer Motion, smooth scrolling with Locomotive Scroll, and interactive hover effects.

## Project Structure
- `pages/` - Next.js pages including the main index.tsx
- `pages/api/` - API routes
- `components/` - React components
- `styles/` - SASS stylesheets
- `public/` - Static assets
- `modules/` - TypeScript type declarations

## Tech Stack
- **Framework**: Next.js 13.5 with TypeScript
- **Styling**: SASS
- **Animations**: Framer Motion, Locomotive Scroll
- **Effects**: Hover Effect library (WebGL-based)
- **Data Fetching**: SWR

## Development
Run the development server:
```bash
npm run dev
```
The site runs on port 5000.

## Production Build
```bash
npm run build
npm start
```
